    Wie baut sich eine HTML Seite auf?


    Ein HTML Dokument besteht im großen und ganzen aus drei verschiedenen Tags :


    <html></html>
    <head></head>
    <body></body>

    Aus diesen Basiselementen ergibt sich das Grundgerüst einer HTML-Seite.

    Das HTML-Tag


    <html></html>

    • Teilt dem Browser mit, das es sich um ein HTML-Dokument handelt
    • Umgibt alle anderen Tags

    Das HEAD-Tag


    <head></head>

    • Text
    • Text

    Das BODY-Tag


    <body></body>

    • Text
    • Text

    Das Grundgerüst

    
<html>

	<head>
		
	</head>	
	
	<body>
	
	</body>
	
</html>
