Was ist Brackets?

  • quelloffener Editor für Webdesign und Webentwicklung von Adobe
  • geschrieben in HTML, CSS & JAVASCRIPT
  • einfache UI, hilfreiche Komfortfunktionen, Live-Editfunktion der Webinhalte
  • Erweiterbar mit neuen Funktionen
  • Noch in Entwicklung

Was kann Brackets?

    Brackets kann CSS, HTML & JAVASCRIPT
  • Live-Vorschau im Browser
  • Code-Vervollständigung
  • JSLint integriert

Gut zu wissen!

  • cmd + e - Quickedit
  • cmd + d - Zeile duplizieren
  • cmd + shift + o- Quickopen Funktion
  • cmd + alt + s - Projekt abspeichern
  • WICHTIG! - Dateien/Ordner immer über "Open Project" öffnen
