Was ist CSS?


    Cascading Style Sheets

    = übergeordnete Stilvorlagen



    - Formatierung von HTML-Elementen


    - Trennung von Struktur und Gestaltung:

    -> HTML sorgt für Struktur
    -> CSS sorgt für Gestaltung
    - aktuell: CSS 3
    seit 2000 in Entwicklung

Chancen


    - mehr Möglichkeiten der Gestaltung
    Bsp.: Schriftgestaltung, Rahmen, Innen- und Außenabstände, ...


    - bessere Aktualisierbarkeit

    - Accessibility / Barrierefreiheit

    - Corporate Design

Risiken


    - Unterschiedliche Unterstützung durch Browser

Was sind Element-Selektoren


    - Tag-Namen wie z.B. p, div oder body kann man damit direkt ansprechen.

     p { color: red; }

    -> sorgt dafür, dass alle p-Tags eine rote Schriftfarbe haben.

Klasse


    Möchte man jedoch nicht alle Tags ansprechen, kann man diesen Klassen zuweisen.

    < p class="inhalt">Text< /p>

    Diese Klasse ruft man dann mit einem Punkt in der CSS auf

     .inhalt{color: red;}

    Wenn man aber mehrere tags mit dem selben Klassen-Namen hat

     p.inhalt{color: red;}

    Man kann auch mehrere Klassen definieren und gleichzeitig verwenden

     < p class="inhalt schriftfarbe schriftart">Text

ID


    Das ganze Funktioniert auch mit IDs. 

     < h2 id="inhalt">Text2< /h2>

    Diese werden durch Rauten im CSS angesprochen 

     #inhalt{color: red;}
