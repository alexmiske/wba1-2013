Was ist CSS?


    Cascading Style Sheets

    = übergeordnete Stilvorlagen



    - Formatierung von HTML-Elementen

    - Trennung von Struktur und Gestaltung:


    -> HTML sorgt für Struktur

    -> CSS sorgt für Gestaltung


    - aktuell: CSS 3

    seit 2000 in Entwicklung

Chancen


    - mehr Möglichkeiten der Gestaltung

    Bsp.: Schriftgestaltung, Rahmen, Innen- und Außenabstände, ...


    - bessere Aktualisierbarkeit
    - Accessibility/Barrierefreiheit

Risiken


    - Unterschiedliche Unterstützung durch Browser

Vererbung


    HTML-Dokument gleicht einer Baumstruktur:

    Vererbungsprinzip gilt auch für CSS:

    body
{
  font-family: Arial;
  color: #fff;
  text;
}
