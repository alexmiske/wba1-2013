Ausblick 1 - XHTML

XHTML (Extensible HyperText Markup Language)

  • XML-gerechtes HTML
  • Definiert durch XML (Extensible Markup Language) statt SGML (Standard Generalized Markup Language)
    • - XML-gerechtes HTML - Definiert durch XML(Extensible Markup Language) statt SGML(Standard Generalized Markup Language) - Syntaktisch kompatibel zu wichtigen XML-Standardsprachen(SVG, WML, SMIL) - strikter als HTML(Code muss „sauber“ sein: End-Tags dürfen nicht vergessen werden wie in HTML)

Ausblick 2 - HTML5

HTML5 (Nachfolger von HTML 4.01 und XHTML 1.0)

  • Fortführend von HTML / Ziel: HTML neu erfinden
  • Anleihen/Standards aus XHTML
  • In Entwicklung, soll 2014 laut W3C offiziell verabschiedet werden
  • Viele neue Features, wie z.B. Audio/Video Support (macht Plugins wie Adobe Flash überflüssig)
